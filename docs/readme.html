HTML Structure

This theme is a responsive layout with one columns. All of the information within the main content area is nested within a sections with an class of "content". The general template structure is the same throughout the template. Here is the general structure.

<!-- Site Header -->
<header id="site_header">

    <!-- Content Area -->
    <div class="container">

        <!-- Section -->
        <section class="content">

        </section>
        <!-- Section-->

    </div>
    <!-- Content Area -->

</header>
<!-- Site Header -->

<!-- Site Main -->
<main id="site_main">

    <!-- Content Area -->
    <div class="container">

        <!-- Section -->
        <section class="content">

        </section>
        <!-- Section -->

    </div>
    <!-- Content Area -->

    <!-- Content Area -->
    <div class="container">

        <!-- Section -->
        <section class="content">

        </section>
        <!-- Section -->

    </div>
    <!-- Content Area -->

</main>
<!-- Site Main -->

<!-- Site Footer -->
<footer id="site_footer">

    <!-- Content Area -->
    <div class="container">

        <!-- Section -->
        <section class="content">

        </section>
        <!-- Section - Contact -->

    </div>
    <!-- Content Area -->

</footer>
<!-- Site Footer -->

If you would like to edit the color, font, or style of any elements in one of these columns, you would do the following:

.hc-menu li a {
    color: #someColor;
}

If you find that your new style is not overriding, it is most likely because of a specificity problem. Scroll down in your CSS file and make sure that there isn't a similar style that has more weight. I.E.

.hc-menu li a:hover {
    color: #someColor;
}

So, to ensure that your new styles are applied, make sure that they carry enough "weight" and that there isn't a style lower in the CSS file that is being applied after yours.

CSS Files and Structure

I'm using five CSS files in this theme. The first one is a generic reset file. Many browser interpret the default behavior of html elements differently. By using a general reset CSS file, we can work round this. This file also contains some general styling. Keep in mind, that these values might be overridden somewhere else in the file.

The second file contains all of the specific stylings for the page. The file is separated into sections using:

/**********************************************************/
/*** MAIN LAYOUT ******************************************/

some code

some code

/*-- Containers ------------------------------------------*/

/*-- Site --*/

/*-- Main --*/

some code

some code

/**********************************************************/
/*** BASE UI **********************************************/

/*-- Input -----------------------------------------------*/

some code

/*-- Textarea --------------------------------------------*/

some code

/*-- Button ----------------------------------------------*/

some code

/*-- Transitions -----------------------------------------*/

some code

etc, etc.

If you would like to edit a specific section of the site, simply find the appropriate label in the CSS file, and then scroll down until you find the appropriate style that needs to be edited.

Any images that are placed within the blog section have 1px border, 4px worth of padding and a light grey background. If you would like to edit the display of these images, find the following section in the style sheet:

.features {
    change styles here:
}

.features img {
    change styles here:
}

JavaScript Files

This theme imports fifteen JavaScript files.

  1. jQuery 1.10.2 - jQuery JavaScript Plugin
  2. HTML5 Shiv 3.6.2 - Enables use of HTML5 sectioning elements in IE 6-8
  3. Respond 1.1.0 - CSS3 Media Queries for IE 6-8
  4. Selectivizr 1.0.2 - Emulates CSS3 pseudo-classes and attribute selectors in IE 6-8
  5. Easing 1.3 - Plugin to give advanced easing options
  6. hcSticky 1.1.95 - Makes elements on your page float as you scroll
  7. ScrollTo 1.4.6 - Easy element scrolling
  8. One Page Nav 2.2.0 - Navigation on one-page sites
  9. Lightbox 2.6 - Used to overlay images on top of the current page
  10. ElasticSlideshow 1.0.0 - Simple elastic slideshow with a thumbnail preview
  11. Tweet.js Mod 1.4.0 - Put twitter on your website with tweet
  12. GMaps 0.4.4 - Adds Google Map on your website
  13. Animate 1.0.0
  14. Contact Form 1.0.0
  15. Settings 1.0.0

jQuery is a Javascript library that greatly reduces the amount of code that you must write. Most of the animation in this site is carried out from the customs scripts. There are a few functions worth looking over.

$(".scroll-gallery").sliderkit({
    auto:false,
    shownavitems:3,
    scroll:1
});
